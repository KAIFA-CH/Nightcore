# Nightcore AdonisJS DB site
##### Created by Mar0xy - Published via MXY Group

## Setup
configure .env
run `npm install`.


### Migrations

Run the following command to run database migrations make sure you have adonisjs installed.

```js
adonis migration:run
```


### Start Server
run `node server.js`.
